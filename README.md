# KubeSync DevOps Platform

A production-grade DevOps pipeline with real-time infrastructure monitoring.

## Features
- Automated CI/CD pipeline using GitHub Actions
- Dockerized Python web application
- Real-time monitoring with Prometheus and Grafana
- Hosted on AWS EC2

## Tech Stack
- Python, Docker, GitHub Actions, AWS EC2, Prometheus, Grafana
